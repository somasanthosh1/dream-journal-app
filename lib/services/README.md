# 🌙 Dream Journal App

A Flutter mobile application to record, manage, and revisit your dreams with Firebase cloud storage.

---

## 📱 About the App

Dream Journal App is a mobile application built using Flutter and Firebase. It allows users to quickly record their dreams with a title, description, date, and mood tag. All dream entries are stored securely in Firebase Firestore cloud database.

---

## ✨ Features

- 🌙 Add dreams with title, description, date and mood tag
- 📋 View all dreams in a scrollable list
- 🔍 Search dreams by keyword in real time
- 📖 View full dream details
- ✏️ Edit existing dream entries
- 🗑️ Delete dream entries
- ☁️ Firebase Firestore cloud storage
- 🎨 Beautiful dark night themed UI

## 🚀 How to Run This Project

1. Clone this repository
2. Run flutter pub get
3. Run flutterfire configure
4. Run flutter run -d chrome
